# Nonogram console program

Hello, this is my first repository on Github, so the way that this repository looks may not be the best. I decided to
challenge myself with a project to create a Nonogram program in the C language. If you see this project and feel like
you can make some improvements, feel free to let me know.

## Goals

- Have a Nonogram game running in the console
- Create, save, edit, and open nonogram puzzle files in the program
- Have an easy to read display to see the location of each tile
- Have a puzzle generator
- Change the resolution of the console window
- Have various size nonogram puzzles
- Have portable code for Windows, Mac, Linux, etc.
- Learn programming concepts from this project
